# Installation

Install OPLINK using npm:

```bash
npx oplink@latest init
```

## Cursor Configuration

If using JSON in Cursor to setup your MCP server:

```json
{
  "mcpServers": {
    "workflows-mcp": {
      "command": "npx",
      "args": ["oplink@latest server"]
    }
  }
}
```

## Additional Options

To add specific workflows or presets:

```bash
# Adding specific workflows
npx oplink@latest add thinking
npx oplink@latest add debugger-mode

# Adding multiple workflows
npx oplink@latest add task-manager vibe-coder react
```
